# Ghost
最大4人で遊べる鬼ごっこのゲームです。このゲームには1人のGhostと1～3人のEscapeeがいます。GhostはすべてのEscapeeを捕まえれば勝ちです。EscapeeはGhostをライトで照らし、体力を0にしたら勝ちです。勝敗が付かないままタイマーが0になったら引き分けです。このゲームの大きな特徴はプレイヤー自身が遊ぶステージを作成できることです。また、LLMにプロンプトを渡すことで要望に沿ったステージを自動で作成させることが出来ます。

### Game
プレイヤーはJoyConまたはキーボードで遊ぶことが出来ます。  
Escapeeは持っているライトでGhostの体力を0にすることを目標とします。ライトにバッテリーがあり、付けていると消耗します。ステージに配置された電池を補充することでバッテリーを回復しましょう。Escapeeは暗闇の中では見ることが出来ません。位置を確認する方法は、自分のライトを照らすか、一定時間ごとに点滅する鼓動から推測する、またはステージに配置された明かりの下に移動するしかありません。Escapeeの鼓動の周期はGhostとの距離に応じて変わります。  
GhostはすべてのEscapeeを捕まえることを目標とします。Escapeeからのライトか、ステージに配置されたPatrolからライトを照らされたらHPが減ってしまいます。GhostはEscapeeとは別で基本的にはステージに表示されます。しかし、JoyConならRボタンキーボードならマウスの左クリックまたは→キーを押すことで、透明になることが出来ます。うまく透明化を使ってEscapeeをとらえましょう。

### Stage
マウス操作でステージを作成することが出来ます。ステージは複数作成し、保存することが出来ます。
* Block: 障害物となるブロック。
* Door: 障害物となるドア。Keyを持っていると開くことが出来る。
* Patrol: ステージを巡回するパトロール。ライトをもち、Ghostの体力を減らすことが出来る。
* Key: DoorとTreasureを開けることが出来る。
* Battery: Escapeeの持つライトのバッテリーを補充するアイテム。
* Treasure: Key、Batteryを入れることが出来る。
* Candle: ステージのどこでも配置できる。ステージを照らすアイテム。  

![](App/Rule/create.png)

LLMにプロンプトを渡してステージを作成させることが出来ます。プロンプトは英語の方が精度が良いです。また巻ている間はブロック崩しが出来ます。ブロックの代わりに表示されている文字を崩します。

### 開発
描画と入力はSiv3Dのライブラリを使っています。  
LLMのステージ自動作成では、ステージのレイアウトと各オブジェクトの詳細パラメータを一斉に生成させると、配置制約やパラメータ制約などの条件に違反するケースが頻発しました。そこで、ステージ生成を2段階に分け、1段階目でレイアウトのみ、2段階目で各セルのパラメータを生成させることで、LLMが扱う問題を分離し、制約違反を大幅に減らすことができました。さらに、出力結果を検証し、制約に違反している箇所があれば、そのエラー内容をLLMにフィードバックとして再入力するループを設計しました。それでも完全に制約違反を防ぐことはできなかったため、最終的にはルールベースで自動修正する仕組みを追加し、プレイヤーが必ず遊べるステージを生成できるようにしました。
